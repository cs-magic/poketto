## 配置

### 1. 环境变量

参考 `.env.sample` 里的配置一下 `.env`，一些是可选的，一些是必填的，不懂的要去对应平台查询配置使用。

### 2. 初始化

```shell
yarn
yarn prebuild
```

### 3. 开发模式启动

```shell
yarn dev
```

### 4. 生产模式启动

```shell
yarn build && yarn start
```
